PERSONAL DETAILS, three editorial cuts.

Same data, the same navy/terracotta language we just shipped — three ways to lift the section out of the generic "grey filled boxes" look into the Cargo editorial vibe. The left rail stays as-is; only the content panel changes. Pick one (or mix: e.g. A's read view + B's edit inputs).

Option AThe Crew Dossier

Warm paper ground. Reads like a printed crew file: tracked-caps labels, values set in ink, every field underscored by a hairline rule. Empty fields fall back to an italic serif placeholder rather than an empty box. Best "premium document" feel; quietest in edit mode (rules become inputs).

Option BLine Fieldsets

Cool white card on the navy ground we shipped. Fields are grouped into italic-serif sub-sections (Identity · Travel documents · Contact · Health) with a terracotta diamond marker and a trailing hairline. Inputs are underline-only, not boxed — they go terracotta on focus. Most "form-friendly": looks identical in read and edit.

Option CIndex Cards

Each field is its own soft card on the cool ground. Caps label top-left, value below; required fields get a terracotta tick, the focused field gets a terracotta ring + soft halo. The most modern/scannable; reads well on tablet. Slightly busier than A/B — good if you want fields to feel tappable.
