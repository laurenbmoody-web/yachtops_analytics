Rota list view — four ways

Same six crew, same data (Chief Stew & Marco work a split watch; Emma a long single day). Each option is built around one question: when do I start, break and finish today?

Option A · ambient timeline
See the day without clicking.

Every row carries a slim 24-hour bar. Work blocks are solid; the gap between them is the break. Start & finish times sit right on the blocks, so split watches read instantly. Tapping still opens the full rest panel. Best for at-a-glance scanning of a whole department.

Option B · tap to expand
Tidy list, full detail on tap.

Rows stay compact (name · shift summary · hours). Tapping a row slides open an inline panel with a plain-English breakdown — Start · Break · Finish — plus the timeline. No popover, no context switch. Best when the list is long and you only want detail for one person at a time. (Emma's row shown expanded.)

Option C · spelled-out segments
No interpretation needed.

Each shift is broken into labelled chips — Start, Break, Finish — shown right in the row. Split watches list every block. Zero timeline-reading; great for crew who just want the numbers. Costs vertical space, so denser crews scroll more.

Option D · personal cards
Reads like a shift card.

A larger card per crew member: timeline, big start/finish, and a one-line summary in plain English (“Two watches…” / “Straight through, no break”). Most personal — feels like “my day” for a crew member opening the app. Fewer fit on screen, so better for a focused/own-row view than scanning a big department.
