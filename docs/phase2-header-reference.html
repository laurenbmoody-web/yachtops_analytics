Notes for implementation
• The title qualifier ("a charter" / "an owner trip" / "a visit") is type-aware. Charter trips read "a charter", owner trips "an owner trip", "Other" trips "a visit".
• The subtitle is auto-generated from trip state — what's happening now and what's next. Updates as the trip moves.
• Right rail's NOW shows current local time (vessel timezone). ON DUTY reads the current on-duty crew from the rota.
• The 3-cell band: Aboard shows guest count + brief ashore note. Today shows the route's stop-type as serif italic. Coming up highlights the next service moment in terracotta italic.
• In Planning phase: 3-cell band becomes Days out / Guests confirmed / Ready. In Settling: Closed out / Photos / APA final.
